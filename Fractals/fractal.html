Interactive Fractal Generator

Welcome fractal lovers!

This interactive fractal generator will let you use the Mandelbrot, Julia, and other fractals to your choosing. Select a fractal now using the dropdown menu.

You can select constant c values for Julia sets and Dr. Brown's with the text box. Some cool values for Julia Sets are given for you to try! The best values for Dr. Brown's fractal are all positive x values and a y value of 0. The default value is (-.7, .27015).

Use the key values to zoom in and around the fractals with r to reset parameters. To use full screen, try clicking the full screen button!
